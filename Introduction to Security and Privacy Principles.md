## Security Principles

- The core elements of security principles is the CIA triad 
- **Confidentiality** protects data from unauthorized access. Confidentiality can be upheld through the application of the following controls
	- Encryption of data at rest
	- Encryption of data in transit 
	- Encryption of data in use
	- Access Control
- **Integrity** protects the system or data from unauthorized modification. It is supported through the implementation of the following controls
	- [[Hashing]]
	- Digital signing
	- Configuration management
	- Change control
- **Availability** keeps the systems up and data available in a timely manner to authorized users. The following list provides some controls that map to availability
	- Clustering 
	- Load balancing 
	- Data backups 
	- Failover configurations
	- Rollback functions
